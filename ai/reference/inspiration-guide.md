# Inspiration Collection Guide

## Goal

Collect inspiration that helps communicate:

* Clarity
* Trust
* Systems Thinking
* Simplicity
* Long-Term Value

Do NOT collect inspiration simply because it looks cool.

---

# What To Collect

## Hero Sections

Study:

* Layout
* Typography hierarchy
* Messaging structure
* Visual balance

Save screenshots.

---

## Framework Sections

Study:

* Process diagrams
* Systems maps
* Flow charts
* Visual frameworks

These are especially important.

---

## Editorial Sections

Study:

* Large typography
* Reading experience
* Content hierarchy
* White space

---

## Diagram & Information Design

Study:

* Architecture diagrams
* Relationship maps
* Information architecture
* Blueprint-style visuals

---

## Layouts

Study:

* Section composition
* Grid systems
* Content rhythm
* Visual hierarchy

---

## Navigation

Study:

* Simplicity
* Clarity
* Information structure

---

# Do NOT Collect

Avoid collecting inspiration primarily because of:

* Fancy animations
* 3D graphics
* Visual effects
* Trendy aesthetics
* Startup hype visuals

These are secondary.

---

# Questions To Ask

For every screenshot:

1. What specifically do I like?
2. Why does it work?
3. Does it support my brand?
4. Can I explain it in one sentence?

If not, do not save it.

---

# Desired Feeling

The website should feel:

* Calm
* Intelligent
* Structured
* Thoughtful
* Trustworthy
* Premium

The visitor should think:

"This person thinks clearly."

Collect 10–20 exceptional screenshots and organize them into:

```
hero/
framework/
editorial/
diagrams/
layouts/
```

A small, high-quality inspiration library is far more useful than a folder full of random beautiful websites.

# STEP 7 — Screenshot Rule

Every screenshot filename should explain WHY.

Example:

```
hero-large-typography.png

framework-diagram-layout.png

editorial-spacing.png

cta-composition.png
```

Never:

```
cool-site-1.png
```

🤣

# STEP 8 — Stop At 25 Screenshots

When finished, tell me:

```

Top 3 framework sections

```

Right now you're not looking for a homepage.

> The visual language of a trusted systems advisor who helps businesses reduce friction and build stronger foundations.
